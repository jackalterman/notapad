# Notapad — Project State

**Last updated:** 2026-04-14 — B1–B9 all fixed; all enhancements complete
**Phase:** Active Development

---

## Objectives (original)
1. Split `notapad.py` (~1700 lines) into a modular structure under `notapad_app/`.
2. Stabilize the custom menu system (`AntiqueMenu`) by converting it from a `tk.Toplevel` to an embedded `tk.Frame` using `.place()` — eliminating focus loss / windowing bugs on Windows.
3. Decouple dialogs, the code editor, syntax config, and main entry point.

---

## Module Structure
| File | Purpose |
|---|---|
| `notapad.py` | Main orchestrator: `Notapad` class, all UI wiring, file I/O, search logic, zoom, keybindings, `main()` entry point |
| `notapad_app/__init__.py` | Package init (minimal) |
| `notapad_app/config.py` | `APP_NAME`, `THEMES` dict (light/dark), syntax pattern dict `P`, `EXT_LANG`, `LANG_LABEL` |
| `notapad_app/dialogs.py` | `open_find`, `open_replace`, `goto_line`, `choose_font`, `show_about`, `show_appearance_submenu` |
| `notapad_app/editor.py` | `apply_highlight()` — viewport-scoped syntax highlighting via regex patterns from `config.P` |
| `notapad_app/settings.py` | `SettingsManager` — persistent user settings saved to `~/.notapad.json` |
| `notapad_app/ui_engine.py` | `AntiqueMenu` (embedded `tk.Frame` menu via `.place()`), `apply_windows_title_bar` (DWM dark mode) |
| `notapad.spec` | PyInstaller spec for standalone `.exe` build |
| `build/` | PyInstaller build artifacts |
| `dist/` | PyInstaller distributable output |

## Key Architecture Notes
- `AntiqueMenu` is an embedded `tk.Frame` using `.place()` on root — intentionally NOT a `Toplevel` — to prevent Windows focus/windowing bugs.
- `_on_app_click` in `Notapad` walks the widget hierarchy to detect clicks outside the menu and close it.
- `editor.py` highlights only the visible viewport ±40 lines for performance, with a 300,000-char hard cutoff.
- `settings.py` saves to `~/.notapad.json` with a 2-second debounce; `save_immediate()` is called on exit to guarantee flush.
- `SettingsManager` defaults: `theme_mode=system`, `word_wrap=True`, `show_line_nums=True`, `status_bar_visible=True`, `font_family=Consolas`, `font_size=11`, `geometry=None`, `last_file=None`, `last_cursor="1.0"`.
